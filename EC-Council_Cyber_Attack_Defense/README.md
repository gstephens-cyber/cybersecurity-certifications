## Overview
This folder contains proof of completion for the EC-Council Cybersecurity Attack and Defense Fundamentals specialization.  
This certification validates foundational skills across offensive security, defensive security, and digital forensics.

It complements hands-on project work from NAIT, IBM, and Google by strengthening baseline knowledge across the attack-defense lifecycle.

---

## Included Courses

### Ethical Hacking Essentials (EHE)
- Reconnaissance and footprinting  
- Vulnerability identification  
- Password attack fundamentals  
- Social engineering concepts  

### Network Defense Essentials (NDE)
- Secure network architecture  
- Firewall rules and ACLs  
- Intrusion detection and prevention concepts  
- Risk mitigation and security controls  

### Digital Forensics Essentials (DFE)
- Evidence acquisition and chain of custody  
- File system forensics  
- Log analysis and event correlation  
- Incident response methodology
